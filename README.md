# Amana Full-Stack Projects

This repository contains a collection of full-stack applications and assignments completed during a 3-month bootcamp. The projects focus on building and enhancing real-world applications using modern web technologies, including frontend development, backend APIs, database integration, and deployment.

## Projects

### 1. StudyBoost AI
An AI-powered study assistant that generates summaries, flashcards, quizzes, and personalized study plans from user notes.

- Upload PDF, DOCX, or text input
- AI-generated summaries, quizzes, and flashcards
- Personalized study plan based on performance
- Progress tracking and dark mode support  
- Tech: React, TailwindCSS, Node.js, Express, Google Gemini API  
- Live: https://studyboostai.onrender.com/

---

### 2. Amana Bookstore
A full-stack bookstore application with browsing, reviews, and cart functionality.

- Book listings and detailed views
- User reviews and shopping cart
- MongoDB Atlas integration  
- Tech: Next.js, TypeScript, MongoDB, Tailwind CSS  
- Live: https://amana-bookstore-kwxeqfm6b-rjs-projects-2b8d78ca.vercel.app/

---

### 3. Bookstore API (Next.js)
Serverless API for managing books and reviews.

- CRUD operations for books and reviews
- Filtering (top-rated, featured, date range)
- Header-based authentication  
- Tech: Next.js API Routes, Node.js, Vercel  
- API: https://amana-bookstore-next-api-peach.vercel.app/api

---

### 4. Amana Marketing Dashboard
A marketing analytics dashboard with real-time insights.

- Campaign performance tracking
- Demographic and regional analytics
- Interactive charts and filtering  
- Tech: Next.js, TypeScript, Tailwind CSS  
- Live: https://amana-marketing-2386.vercel.app/

---

### 5. Amana Transportation
A monitoring dashboard for bus system operations.

- Overview of system status and activity
- Key metrics (active, maintenance, utilization)
- Incident tracking across buses  
- Tech: Next.js, React, TypeScript  
- Live: https://amana-transportation-blond.vercel.app/

---

### 6. Realtime Chat App
A real-time messaging application using WebSockets.

- Instant messaging between users
- Multi-user support  
- Tech: Next.js, TypeScript, Ably  
- Live: https://chat-81ziib8cv-rjs-projects-2b8d78ca.vercel.app/
